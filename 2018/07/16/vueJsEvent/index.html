阻止事件冒泡

事件冒泡

HTML的事件会按照DOM的结构由内向外冒泡触发，如下所示：

<div @click="show3()" id="app">A3
    <div @click="show2()">A2
        <button @click="show1()">A1</button>
        </div>
</div>

var app = new Vue({ 
    el: '#app',
    data: {
        message: 'Hello Vue!'
    },
    methods:{
        show1:function(){
            console.log(111);
        },
        show2:function(){
            console.log(222);
        },
        show3:function(){
            console.log(333);
        }
    }
});

点击A1之后的输出如下：

111
222
333

普通方式阻止事件冒泡

stopPropagation()方法可以阻止事件冒泡。

将button的事件加上$event参数：


<div @click="show3()" id="app">A3
    <div @click="show2()">A2
        <button @click="show1($event)">A1</button>
        </div>
</div>

使用event参数调用stopPropagation方法。


var app = new Vue({ 
    el: '#app',
    data: {
        message: 'Hello Vue!'
    },
    methods:{
        show1:function(e){
            console.log(111);
            e.stopPropagation();
        },
        show2:function(){
            console.log(222);
        },
        show3:function(){
            console.log(333);
        }
    }
});

输出如下：

111

使用vue的方式阻止事件冒泡

只需在事件后面加上.stop即可


<div @click="show3()" id="app">A3
    <div @click="show2()">A2
        <button @click.stop="show1()">A1</button>
        </div>
</div>

取消默认动作

html标签都有一些默认的行为，比如form的提交，a标签的跳转,
原生js可以使用preventDefault取消默认动作，用法和阻止事件冒泡一样。

vuejs的用法：

只需在事件后加上.prevent即可

<div  id="app">
   <a href="http://www.baidu.com" @click.prevent="show1()">A1</a>
</div>

点击A1标签不会跳转到百度。

只触发一次

<div  id="app">
   <a href="http://www.baidu.com" @click.once="show1()">A1</a>
</div>

多次点击A1标签，show1也只会被触发一次。
