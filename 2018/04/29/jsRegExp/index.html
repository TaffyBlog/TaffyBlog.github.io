使用方法

var filter = /^[a-z]at$/;

var value = 'cat';
if (filter.test(value)) {
    console.log('ok');
}

//或者

var filter = new RegExp('^[a-z]at$');
var value = 'cat';
if (filter.test(value)) {
    console.log('ok');
}

//输出ok

\b \B

  1. \b匹配一个单词的边界 \B匹配一个单词的非边界

  2. 表达式 \bhi\b 可以匹配 hi this 表示找出的hi两边必须有空格

  3. 表达式 \Bhi\B 能匹配 hi this 表示找出的hi两边必须没有空格

\d \D

  1. \d匹配一个数字字符 \D匹配一个非数字字符

  2. 表达式 \d 能匹配 123456

  3. 表达式 \D 能匹配 A a

\w \W

  1. \w匹配数字、字母、下划线 \W匹配非数字、字母、下划线

  2. 表达式\w能匹配123ab_

  3. 表达式\W能匹配@#$%

^ $

^表示字符串开头 $表示字符串结尾

[]

  1. []匹配指定一堆字符中的一个
    例如：
  2. [a - z]表示a- z中的其中一个字符。
  3. [A - Za - z]表示a- z或A - Z中的其中一个字符
  4. [aeiou]表示aeiou这几个字符中的其中一个
  5. [^xyz]表示不匹配xyz这三个字符。

+

  1. +表示前一模式可以被重复1次或者n次
    例如：
  2. 表达式 ^go+gle$ 能匹配 gogle google gooooooogle
  3. 表达式 ^g[aeiou]+gle$ 能匹配 gagle gegle geagle geaeaioueagle

* ? {x} {x,y} {x,}

  1. 号表示前一模式可以被重复0次或n次，和+号的区别就是+必须重复一次，可以不重复。

  2. ?号表示前一模式可以被重复0次或1次。要么出现1次要么不出现

  3. {x}表示前一模式可以被重复x次

  4. {x,y}表示前一模式可以被重复x到y次

  5. {x,}表示前一模式必须至少重复x次。

()

  1. ()可以指定一堆字符来匹配一个模式
    例如：
  2. ^(very )*large$ 可以匹配 large very very large 等

|

|用来指定几个规则只要匹配一个规则即可完成匹配
例如： ^abc.(com|cn|net)$ 能匹配 abc.com abc.cn abc.net

\

\转义 ，如果要匹配{、^、+等有特殊含义的字符，可以用\转义。
例如: ^1*(2+3)=6$ 只能匹配 1*(2+3)=6
